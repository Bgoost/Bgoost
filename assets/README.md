# Assets for profile README

Add images here to use in your GitHub profile README:

- **`banner.png`** — Header banner (e.g. 1200×320 px). Uncomment the banner line in the main README to show it.
- **`photo.png`** — Optional profile photo for an "About" section. Reference it in the README with:
  ```markdown
  <img src="https://raw.githubusercontent.com/Bgoost/Bgoost/main/assets/photo.png" width="120" alt="Cristina Manzanares" />
  ```

After adding files, push to the `main` branch so the links work.
