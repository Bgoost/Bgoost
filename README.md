### Project overview (short)

| Project | What it is |
|--------|------------|
| **libft** | C library with reimplemented libc functions (strings, memory, lists). |
| **born2beRoot** | Linux system administration: SSH, sudo, firewall, partitioning. |
| **ft_printf** | Reimplementation of `printf` with format handling. |
| **get_next_line** | Read a file line by line using a single fd. |
| **push_swap** | Sort data using two stacks and a limited set of operations. |
| **fractol** | Fractal viewer (Mandelbrot, Julia, etc.) with minilibx. |
| **pipex** | Reproduce shell pipes and redirections (`< cmd1 \| cmd2 >`). |
| **philosophers** | Dining philosophers with threads and mutexes. |
| **minishell** | Unix shell in C (pipes, redirections, builtins). |
| **netpractice** | Network addressing and subnetting exercises. |
| **cub3D** | 3D raycasting game (Wolfenstein-style) in C. |
| **cpp0–4** | C++ modules 00–04: OOP basics, polymorphism, templates intro. |
| **ft_irc** | IRC server in C++ (RFC 1459). |
| **inception** | Multi-container Docker setup (WordPress, DB, etc.). |
| **cpp5–9** | C++ modules 05–09: templates, containers, algorithms. |
| **transcendence** | Full-stack web project (42 final project). |
